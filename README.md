# 📚 EZ-BookReturn | 二手書收購平台

這是一個專為校園設計的二手書收購預約網站，幫助學生快速估算二手書價值，並輕鬆完成線上預約面交與提問。本專案透過純前端技術打造，並巧妙串接 Google 表單作為輕量級的後台資料管理系統。

## ✨ 核心功能 (Features)

* **💰 估價計算機**：透過直覺的加減按鈕，自動計算「全新」、「良好」、「差強人意」等不同書況的預估收購總額。
* **📅 線上預約**：提供完整的表單防呆機制（檢查學號長度、數量等），填寫完畢後，預約資料將自動同步寫入 Google 表單。
* **❓ 常見問題 (Q&A)**：收錄同學最常問的收購標準、付款方式與面交規則。
* **✉️ 聯絡我們**：專屬的問題回報管道，支援修改時間或取消預約，同樣串接獨立的 Google 表單即時接收學生訴求。

## 🛠️ 使用技術 (Tech Stack)

* **前端架構**：HTML5, CSS3, Vanilla JavaScript (純原生語法，輕量快速，無依賴外部框架)
* **UI 設計**：自適應網頁設計 (RWD)，支援手機與電腦版瀏覽，並引入 Google Fonts (Noto Sans TC) 提升閱讀體驗。
* **後端/資料儲存**：Google Forms (透過 Fetch API `no-cors` 模式背景寫入資料)
